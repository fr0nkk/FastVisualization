Text

Creating a text object

f = fvclear;
fvText('Hello world!','Color',[0 0.5 1],'TextSize',50);
f.Snapshot;

Attach text to object

By default, text is normal to the camera and its size is defined in pixels
pc = fvPointcloud;
fvText(pc,'Test','TextSize',20).Translate([3 3 0]); % Attach text to the corner at [3 3 0]
pc.Rotate([25 -20 0],true);
f.Camera.Translation(3) = -16;
f.Snapshot;

Make 3D text

Set TextSize to 0 to make it 3D, its size will be 1 world unit. You can then modify it with model functions (scale, rotate etc)
fvText('Hello world!','Textsize',0);
 
% Make the view control 3D. (Since the text only occupies the 2D space, the
% fvFigure auto sets the camera constraints to 2D)
f = gcfv;
f.CameraConstraints = '3D';
f.ResetCamera;
f.Snapshot;

Using other fonts

% view list of installed fonts by calling with str2shape without input arguments
available_fonts = str2shape;
 
fvText('Hello world!','Font','Courier New','TextSize',0);
f.Snapshot;

Unicode special symbols also works

fvText(char(9822),'TextSize',0,'Color',[1 1 1]); % display chess knight which is unicode 9822
f.Snapshot;
